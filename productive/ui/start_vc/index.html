Start Version Control¶

PyCharm’s Project tool is a very smart folder browser. We’ll show this by putting our Todo app under version control and seeing how PyCharm can do work for us, with colors indicating VCS status.

Steps¶

  1. VCS -> Import into Version Control -> Create Git Repository.

  2. In Project tool window, right-click on the epc.py file.

  3. At the moment, it is red, meaning, not under version control.

  4. In context menu, choose Git -> Add to VCS.

  5. The color changes to green, indicating it is added but not committed.

  6. In Project tool window, right-click on the root folder.

  7. Choose New -> File.

  8. Provide requirements.txt and click Ok.

  9. In the next dialog, accept adding the new file to VCS.

  10. You likely will get prompted about a Missing .gitignore file in Git project. Click the Create .gitignore link.

  11. Click the Generate button and choose Generate in the button menu.

  12. In the message about adding unversioned files, click the Cancel link.

  13. In requirements.txt, add a line for flask-sqlalchemy. The file should look like this:

    requirements.txt for Start VC section¶
    flask-sqlalchemy

  14. Close requirements.txt by clicking the x on the tab.

  15. You will now be prompted with Package requirement 'flask-sqlalchemy' is not satisfied.

  16. Click the Install requirement link.

  17. Commit these changes to VCS with VCS -> Commit Changes.

  18. Type First commit in the Commit Message box.

  19. Click the Commit button, then choose Commit from the button menu.

  20. Note that the two files in the Project tool window are now a normal color.

Analysis¶

We did a lot of small steps in this task, showing how the tool windows can be put to work:

  • Visual cues. Making a local Git repository caused the project tool to start giving us helpful visual information.
  • Tool context menus. Entries in the project tool can be right-clicked on to get context sensitive menus.

We also used some other features of PyCharm:

  • Install packages. Changing the requirements file prompted for missing packages, then did the work to install.
  • Git. PyCharm is a great visual front-end for version control systems (VCS).
