Rename View¶

Our view function is named hello_world. Let’s change it to home_page. Along the way, let’s get introduced to some of the UI elements in PyCharm.

Steps¶

  1. Close all the editor tabs by clicking the x for each.
  2. Click the red button in the left side of the Run window at the bottom.
  3. Click the 4. Run tab button in the bottom of the screen to close that tool window.
  4. Click the 1. Project tab button in the left edge of window to collapse the Project Tool Window.
  5. Click it again to re-open.
  6. In the Project Tool Window toolbar, click the TODO Image collapse button.
  7. Click the epc folder to expand/open it.
  8. Double-click the epc.py file to open it in the editor.
  9. Click somewhere inside the symbol hello_world, putting the caret in that function name.
  10. From the PyCharm menu, choose Refactor -> Rename.
  11. In the dialog box, change hello_world to home_page and click Refactor.
  12. In the next prompt, click the Do Refactor button.
  13. Afterwards, your epc.py file should look like this:
  14. Reload your browser to confirm all is well.

Analysis¶

In this task we made a small change, but used that change to explore much of the PyCharm UI.

  • The editor has tabs which can be closed (plus lots more that we’ll see later.)
  • The IDE has “tool windows” that are placed on the left, bottom, and right
  • These tool windows can be toggled open and closed
  • Inside the window, each tool has its own little UI (toolbars, tabs, etc.)
  • The Project tool is pretty important
